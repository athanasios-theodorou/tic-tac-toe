# Tic-Tac-Toe

A modern Tic-Tac-Toe web application built with React, focusing on clean component architecture, predictable state updates, and structured game logic.

## Demo 

[![Play Live](https://img.shields.io/badge/PLAY-LIVE%20DEMO-yellow?style=for-the-badge)](https://tic-tac-toe-by-theod.vercel.app/)

## Features

- Two-player gameplay (X / O)
- Dynamic player name editing
- Turn-based state management
- Win and draw detection logic
- Move history tracking
- Game reset functionality

 ## Installation

Install dependencies and start the app:

```bash
npm install
npm run dev
```

## Technologies

- React (Hooks)
- JavaScript (ES6+)
- HTML5
- CSS3
